Lesson 1: Adding CSS Style To HTML



Internal CSS STYLE

    Style Directly Inside HTML Code

  • Add Style Tags Directly Inside Your Tag By Opening a Tag and Adding the 'STYLE' Attribute
  • Type out your style and close the HTML tag
  • Example: [pTag style = "color: #222; text-align: center;">/p

External CSS STYLE

    Link To .CSS File

  • Create a file inside your root folder called main.css
  • Link main.css to your HTML Document inside the HTML File between the opening and closing head tag
  • Link: [link rel="stylesheet" href="main.css"]
  • Now the selectors you choose on your .css file will change the style on your HTML page

Linking CSS Either Internal or External takes time to learn!
When done properly and concise your HTML webpage will have a beautiful new look... A look that will 'WOW' you and those landing on your site...
